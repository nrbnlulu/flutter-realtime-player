### Flutter realtime player

This is a library that focuses on realtime streams support for flutter using ffmpeg.

### Architecture
- ./rust - utilizes ffmpeg to create video streams, irondash_texture is then used to pass the pixel buffers to flutter.
- ./rust/api - dart-rust interop using `flutter_rust_bridge_codegen`
- ./lib/rust - everything under here is autogenerated.
- ./rust/src/core/output - output domain; `flutter_pixelbuffer` owns texture creation/teardown and bridges input events to Flutter/Dart.
- ./rust/src/core/input - input domain emits events and accepts commands via channels (resize/seek/terminate).

### commands
- most commands are in `./Taskfile.yml`
- `task codegen` - generates rust bindings to dart, use this when you changed the api.
